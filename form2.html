Text Field Attributes
Text Field Purpose: Collect Single Line Data. For ex. (Name, Address, Email, etc)

<input type="text">
name Names a specific <input> element
Values: text

<input type="text" name="name" >
Autofocus Specifies that an <input> element should automatically get focus when the page loads. Cursor begins here.
Values: 'autofocus'

<input type="text" autofocus="autofocus" >
Autocomplete Specifies that an <input> element should have autocomplete enabled
Values: 'on', 'off'

<input type="text" autocomplete="on" >
